Measurements from the REW

Measure each channel with the played sweeps (preferably multiple times and at multiple locations), measure LP last and leave the mic at this position. Will be easyer for further tests after calibration

AVR settings

  1. Use one way to clear actuals settings:
    1. use odd program
    2. import flat correction curves with MQX saved flat file
  2. Adjust volume level to 0,62 or -18dB (should gives around 75db at Front Right)
  3. (Optionnal, depends on you preference) If you want to use Dynamic EQ. Set Audyssey Dynamic EQ ON, Set the strength of DEQ to 5dB, all other Audyssey settings off.
  4. Disable loudnees management
  5. Sub Measurements:
    1. set LFE LPF to 250hz, then use "revert AVR LPF" to cancel the filter effect
    2. Assign XLR sub output to center to bypass LFE Low Pass filter
  6. Turn off ECO Mode
  7. Check :
    • all distance to 3m (do not use ‘set default’ button)
    • all levels to 0db
